Getting Started

This is the comprehensive style guide intended towards the use of the RED Arena internal web application.

All necessary elements towards the app's construction can be linked on the left navigation.

General Information

  • The side navigation of this page links to more specific guides for individual elements.

  • The guide is best understood when referenced and read all at once.

  • Colors have specific names that are referenced throughout the guide. These names can be referenced in the Colors section of the guide.

    • CSS Construction

  • SASS is used for the page's construction. Specifically SCSS.

  • Nesting should be used when appropriate.

  • All appropriate elements are defined as variables when appropriate. They are located in their appropriate .scss files (ex. _colors.scss). These files are then imported into the main SASS folder.

  • It is recommended that the site is designed "mobile first."
