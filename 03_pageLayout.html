Page Layout

The page should be laid out in a well-constructed and easy to navigate manner. The page should include a grid and be responsive.

Grid

The grid provides an easy, organized, responsive site. The app should use a 12 column grid with a 2% gutter width.


The grid construction in SCSS should involve variables and mixins. The HTML should be constructed with rows and columns so that the grid operates well and with clear organization throughout the code.

Breakpoints

The app should be responsive so that it is easily and clearly legible across many platforms. Breakpoints will be used to implement that responsivity. The breakpoints can be nested in each defined class in the SCSS.

  • Desktop: 900px

  • Tablet: 650px

  • Mobile: Less than 650px
