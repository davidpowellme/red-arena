Page Layout

The page should be laid out in a well-constructed and easy to navigate manner.

Grid

The grid provides an easy, organized, responsive site. The app should use a 12 column grid with a 2% gutter width.


Grid Code                                                                                                         
    $tot-width: 100%;
    $num-col: 12;
    $gut-width: 2%;
    $ind-col-width: ($tot-width - ($gut-width * ($num-col - 1))) / $num-col;
    @mixin grid($geezy){
    width: ($geezy * $ind-col-width) + ($gut-width * ($geezy - 1));
    margin-right: $gut-width;
    &:last-child {
    margin-right: auto;
    }
    }
