Colors

The color palette for the app was chosen in order to convey friendliness while maintaining the professional look of a company web page.

Primary Color

#D9181F
It was important for the color red to be included in the app, given the company name. The red used conveys a warm, rich tone without being too hurtful on the eyes. It is to be used in the main header, borders, links located on the main pages and form modals, and filler for secondary hover buttons. It is also to be used on the header for form element modals.

Secondary Color

#443939
The Secondary Color is intended to complement the Primary Color without taking over. Indeed, its role should be complementary, and used on smaller headers in the app. It is also used for Collapsable Content on the tablet and mobile sizes of the app.

Content Color

#F2F2F0
The Content Color is intended to be the main background color for the app. It is to be used as the background on all app pages and form element modals, except for certain times when it is appropriate for Card Color to be used. This color is also to be used for font color when text appars on headers and most buttons.

Primary Button Color

#00D6A9
The Primary Button Color is to be the color for all Primary Buttons in the app. The color was chosen as a bright accent to the Primary and Secondary Colors of the site. It is easily seen by the user to call their attention to important buttons on pages and form element modals.

Primary Button Hover Color

#00AB8D
It was important for the color red to be included in the app, given the company name. The red used conveys a warm, rich tone without being too hurtful on the eyes. It is to be used in the main header and for secondary buttons.

Font Color

#47383B
The Font Color is to be used for all font on the site that is located on a page or modal where Content Color is the background color. The color was chosen to present the text in a clear, but not overwheling. Different weight should be provided for the fonts to present visible hierarchy throught the app.

Card Color

#DAEEE8
The Card Color is to help the user visually see the information displayed on cards, profile and schedule pages. Ideally, any information that is diplayed via collapsable content should be called out with Card Color.

Alternate Card Color

#C6EBE3
The Alternate Card Color is to help the user visually see the information specifically on the full schedule page. It should alternate between columns for each time frame.

Passive Color

#DFDFDD
The Passive Color is specifically intended to be used on the schedule page, when an appointment has passed. This allows the user to easily see upcoming appointments without being distracted by past appointments on the schedule.
